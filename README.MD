# Etch-a-Sketch

Projeto de desenho interativo desenvolvido como parte do currículo 
The Odin Project — Foundations.

## Demo
[Ver projeto ao vivo](https://rikkyrs.github.io/Etch-a-Sketch)

## Como usar
- Passe o mouse sobre a grade para desenhar
- Clique em **Mudar tamanho** para definir o número de quadrados por lado (1-100)
- Clique em **Resetar** para limpar o desenho

## Funcionalidades
- Grade gerada dinamicamente com JavaScript
- Cores aleatórias RGB a cada interação
- Botão para redimensionar a grade sem alterar o espaço total
- Botão de reset para limpar o desenho

## Tecnologias
- HTML5
- CSS3 + Flexbox
- JavaScript (DOM manipulation, eventos)
